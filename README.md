# Vehicle Recognition AI

An AI-based vehicle recognition system using computer vision to detect vehicles and read license plates from video streams.

## Features

- Vehicle detection using YOLOv8
- License plate detection
- OCR using EasyOCR
- Vehicle logging system
- Snapshot saving
- SQLite database

## Technologies

- Python
- YOLOv8
- OpenCV
- EasyOCR
- SQLAlchemy

## Project Structure

vehicle_recognition/
├── app/
├── models/
├── run.py
├── requirements.txt

## Installation

pip install -r requirements.txt

## Run

python run.py
